# Steel Muse

The official website for Steel Muse, built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com).

## Project Structure

- `src/pages`: Website pages (Home, Privacy, Terms).
- `src/components`: Reusable UI components.
- `src/styles`: Global styles and Tailwind configuration.
- `public`: Static assets (images, fonts).

## Development

All commands are run from the root of the project, from a terminal:

| Command | Action |
| :--- | :--- |
| `npm install` | Installs dependencies |
| `npm run dev` | Starts local dev server at `localhost:4321` |
| `npm run build` | Build your production site to `./dist/` |
| `npm run preview` | Preview your build locally, before deploying |

## Deployment

This project is configured to deploy to GitHub Pages via GitHub Actions.
